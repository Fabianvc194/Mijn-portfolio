# Azure Data Studio — Shortcut Cheatsheet

---

## Query editing — Writing & editing T-SQL

| Shortcut | Actie | Waarom het tijd bespaart |
|---|---|---|
| `F5` | Query uitvoeren | Meest gebruikte actie — muis overbodig. |
| `Ctrl + Space` | IntelliSense triggeren | Kolom- en tabelnames autocomplete zonder typen. |
| `Ctrl + /` | Regel(s) commentaar aan/uit | Snel blokken debuggen zonder code te verwijderen. |
| `Alt + ↑ / ↓` | Regel omhoog/omlaag verplaatsen | Clausules herschikken zonder cut-paste. |
| `Shift + Alt + ↓` | Regel dupliceren (naar beneden) | Snel een JOIN of WHERE-conditie kopiëren. |
| `Alt + klik` | Meerdere cursors plaatsen | Dezelfde edit op tien aliassen tegelijk — enorme tijdwinst. |
| `Ctrl + Shift + [` | Code-blok inklappen | Subquery verbergen terwijl je outer query schrijft. |

---

## Navigatie — Object Explorer & tabs

| Shortcut | Actie | Waarom het tijd bespaart |
|---|---|---|
| `Ctrl + Shift + P` | Command Palette openen | Toegang tot elke ADS-functie zonder menu's. |
| `Ctrl + P` | Snel bestand/query openen | Direct naar een opgeslagen .sql-bestand zoeken. |
| `Ctrl + Tab` | Wisselen tussen query-tabs | Meerdere query-vensters zonder muis navigeren. |
| `Ctrl + G` | Ga naar regelnummer | In een 500-regel stored procedure direct naar een foutlocatie. |
| `Ctrl + K → Ctrl + S` | Keyboard shortcuts editor | Snelle toegang om shortcuts aan te passen. |

---

## Resultaten & uitvoering — Resultatenvenster & debugging

| Shortcut | Actie | Waarom het tijd bespaart |
|---|---|---|
| `Shift + Win + R` | Resultatenvenster tonen/verbergen | Meer editorruimte wanneer je alleen schrijft. |
| `Ctrl + F` | Zoeken in query | Snel een kolomnaam of alias vinden in grote queries. |
| `Ctrl + H` | Zoeken en vervangen | Tabelnaam refactoren door hele query in één stap. |

> **Tip:** Via `Ctrl + Shift + P` → "upper" / "lower" zet je geselecteerde tekst snel om naar UPPER of lowercase — handig voor SQL-stijl consistentie.
